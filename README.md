## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

```bash
$ docker-compose up --build -d
$ open http://localhost:3001

# api-server log
$ docker logs [CONTAINER] --follow
# look into mysql
$ docker exec -it db-server bash
> mysql -u develop -p
> use develop;
> show tables;
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
